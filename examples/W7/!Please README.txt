
Приветствую,

-В папке Interactive HTML лежат интерактивные карты:
	а) с визуализацией реального и прогнозируемого спроса на такси в выбираемый пользователем момент времени;
	б) с визуализацией временного ряда фактического и прогнозируемого спроса на такси в выбираемой области

для двух моделей - прогноз на один час и на три часа вперед.

К сожалению, ползунки для выбора дня, часа и сдвига экспортировать в html не удалось.
Эта функциональность доступна только в ноутбуке.

-В папке ScreenShorts есть скрины карт для некоторых значений параметров.

-Если решите запустить тетрадку (очень бы мне хотелось...:), то установить дополнительно нужно лишь folium.
Сделать это довольно просто: 

$ pip install folium
or
$ conda install folium -c conda-forge

Все необходимые файлы находятся в папке.

Спасибо за внимание!
